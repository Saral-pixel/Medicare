# Medicare
Medicare frontend website with signup and login UI,with a book an appointment system,doctors and rating!

Medicare 🏥
Medicare is a healthcare website UI built using HTML and CSS. It is designed to simulate a doctor appointment booking platform with sections for doctors, departments, and basic authentication pages.

🌐 Features
Clean landing page UI
Hero section with booking CTA
Top doctors showcase cards
Department browsing section
"How it works" section
Login page UI
Signup page UI
Navigation with search bar and categories
